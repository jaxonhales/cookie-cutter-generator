# ─────────────────────────────────────────────────────────────────────────────
# INSTALL
# ─────────────────────────────────────────────────────────────────────────────
# pip install -r requirements.txt
#
# CadQuery requires a working OpenCASCADE install. The easiest way on all
# platforms is via conda:
#   conda install -c conda-forge cadquery
#
# Or pip (includes pre-built OCC wheels since CQ 2.4):
#   pip install cadquery
#
# ─────────────────────────────────────────────────────────────────────────────
# USAGE
# ─────────────────────────────────────────────────────────────────────────────
# CLI:
#   python main.py drawing.png my_cutter.stl --size 100
#     (scales longest dimension to 100mm)
#
#   python main.py drawing.png my_cutter.stl --dpi 300 --height 40 --wall 1.5
#
# GUI:
#   python main.py --gui
#
# ─────────────────────────────────────────────────────────────────────────────
# PACKAGING WITH PYINSTALLER
# ─────────────────────────────────────────────────────────────────────────────
# 1. Install:
#      pip install pyinstaller
#
# 2. Build:
#      pyinstaller --onefile --windowed main.py \
#        --collect-all cadquery \
#        --collect-all OCC \
#        --hidden-import skimage.morphology \
#        --hidden-import scipy.interpolate
#
#    --onefile:   single executable
#    --windowed:  no console window (use --console for debugging)
#
# 3. Output:
#      dist/main          (Linux/Mac)
#      dist/main.exe      (Windows)
#
# ─────────────────────────────────────────────────────────────────────────────
# NOTES ON CADQUERY + PYINSTALLER
# ─────────────────────────────────────────────────────────────────────────────
# CadQuery bundles OpenCASCADE native .so/.dll libs which PyInstaller needs
# to collect. The --collect-all flags above handle this.
# If you hit missing DLL errors on Windows, also add:
#   --collect-all cadquery_occ
#
# Final binary size is typically 150-250MB due to OCC native libs.
# This is normal and expected for CAD software.
#
# ─────────────────────────────────────────────────────────────────────────────
# PHOTO TIPS FOR BEST RESULTS
# ─────────────────────────────────────────────────────────────────────────────
# - Draw on bright white paper with a black sharpie
# - Photo in good lighting with no shadows across the drawing
# - Keep camera parallel to paper (no perspective skew)
# - Use --size to specify the real-world output size rather than relying on DPI
# - Closed shapes (loops) produce cookie cutters with base flanges
# - Open paths produce a swept wall without a flange
